# blueberry-recipes
A small blueberry recipes test app.

To use:
Load the files in Android Studio.
Build with Gradle.
Run the app.
Tap the "Start" button to start.
Tap the "View Sources" button to view the source of the recipe.

Requires:
Android 6.0
